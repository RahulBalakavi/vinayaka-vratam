# Sri Vinayaka Vratam — A Guided Course

A self-contained, narrated slide-deck that walks a devotee through the **Sri Vinayaka
Vratam (Vinayaka Chaviti / Ganesh Chaturthi)** puja, step by step — hear each mantra
recited in Telugu while reading along.

**Live:** https://rahulbalakavi.github.io/vinayaka-vratam/

21 slides: cover → preparation → opening rites → main worship (with the **Ekavimsati
Patra Puja**, the 21 leaves, and the **108 names**) → offerings (with the **Ganapati
Atharvashirsha** as Mantrapushpam) → the Katha (the moon's curse and the Syamantaka
jewel) → conclusion. Each mantra is shown in Telugu script with IAST romanization and a
play-along reciter (Sarvam AI — Karun voice).

Mantras are traditional liturgy transcribed from established repositories
(sanskritdocuments.org, vignanam.org, astrodisha.com); the Katha is a faithful
original-words retelling; images are Creative-Commons / public-domain with attribution.
`[verify]` flags mark spellings and recension choices a knowledgeable reciter should check.
